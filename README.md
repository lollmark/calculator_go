# Калькулятор - Веб-сервис для вычисления арифметических выражений

## 📋 Описание проекта
Данный проект представляет собой веб-сервис для вычисления арифметических выражений.  
Пользователь отправляет HTTP POST запрос с выражением, и сервис возвращает результат вычислений.  

---

## 🚀 Запуск проекта
1. Убедитесь, что у вас установлен Go версии 1.18 и выше.  
2. Клонируйте репозиторий:  
   ```bash
   git clone https://github.com/lollmark/calculator_go.git
   ```  
3. Перейдите в директорию проекта:  
   ```bash
   cd calc_go
   ```  
4. Запустите сервер:  
   ```bash
   go run ./cmd/calc_service/...
   ```  

Сервер запустится на порту `8080`.  

---

## 🛠️ API Эндпоинты
### POST /api/v1/calculate
**Описание:**  
Принимает на вход JSON с арифметическим выражением и возвращает результат.  

**Пример запроса:**  
```bash
curl --location 'http://localhost:8080/api/v1/calculate' \
--header 'Content-Type: application/json' \
--data '{
  "expression": "2+2*2"
}'
```  

**Пример успешного ответа:**  
```json
{
  "result": 6
}
```  

**Ответ с ошибкой (некорректное выражение):**  
```json
{
  "error": "Expression is not valid"
}
```  

---

## 📌 Примечания
- В случае ошибок будет возвращён HTTP статус:  
  - `422 Unprocessable Entity` — при ошибке в выражении  
  - `500 Internal Server Error` — при других внутренних ошибках
